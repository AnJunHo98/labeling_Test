Temporal Frame Labeler

Dataset 폴더 또는 비디오를 불러와 start/end frame 구간을 고르고, v41 라벨을 체크해 JSON/CSV로 저장합니다.

1. 데이터/비디오 불러오기

LeRobot dataset 폴더를 선택하면 meta/info.json, tasks.jsonl, videos/.../episode_*.mp4를 자동 인식합니다.

2. 프레임 확인

현재 frame: 0 / ? 현재 시간: 0.000s

단축키: [ start, ] end, ←/→ 1 frame 이동, Shift+←/→ 10 frame 이동, Enter 구간 저장, Space 재생/일시정지

3. 구간 라벨링

팔 / 행위자
행동 primitive
대상 종류 target_type
물체/관절 운동 object_articulation
세부 관계 상태 relation_state

4. 등록된 구간

0개
# Frame 팔 Primitive Target Articulation Relation Entity hint 메모 관리
아직 등록된 구간이 없습니다.

5. 누적 다운로드 묶음

0개

여러 dataset/episode를 라벨링한 뒤 현재 결과를 누적하면, 마지막에 하나의 bundle JSON/CSV로 한 번에 다운로드할 수 있습니다.

아직 누적된 annotation이 없습니다.

정적 MVP입니다. 결과는 브라우저에서 다운로드되며 서버로 자동 업로드되지 않습니다.
